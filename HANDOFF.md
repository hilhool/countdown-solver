## Что уже сделано и закоммичено (3 коммита в master)

1. `088c404` — рефакторинг пайплайна, verified sampling 93.5%, обновлён README.
2. `59a5b73` — **улучшения данных и обучения** (главное в этом этапе):
   - **`1_generate.py`**:
     - `collect_reachable` теперь хранит **кратчайший канонический** трейс на
       таргет (а не первый из DFS). Коммутативные шаги — больший операнд первым
       (`50 * 3`, не `3 * 50`). Низкоэнтропийная цель → greedy/beam воспроизводит
       надёжнее.
     - Выбор таргетов **взвешен по длине трейса²** (Efraimidis-Spirakis,
       `key = u^(1/w)`), иначе данные заливаются тривиальными 1-шаговыми примерами.
     - Перевес тяжёлого хвоста: `NUM_COUNT_WEIGHTS = {3:0.10, 4:0.30, 5:0.32, 6:0.28}`.
     - Фикс диапазона: `randint(1, 100)` (в тесте есть 100, трейн его не видел).
   - **`2_train.py`**: `EPOCHS = 3` (было 2); лучший чекпойнт восстанавливается.
